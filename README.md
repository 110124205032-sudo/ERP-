# EduTech Portal

EduTech Portal is a comprehensive educational management system designed to streamline the academic experience for both students and teachers. The portal provides distinct dashboards tailored to the specific needs of each user role.

## Features

### Role-Based Access
- Secure login system with specific roles for **Student**, **Teacher**, and **Admin (HOD)**.

### Student Dashboard
The student dashboard provides a centralized view of all academic activities:
- **Overview Dashboard**: Quick access to today's timetable, recent notices, and student profile details including attendance percentage.
- **Full Timetable**: View the complete weekly schedule for all enrolled subjects.
- **Attendance Tracking**: Monitor subject-wise attendance with detailed statistics and status indicators (Good, Warning, Excellent).
- **Study Materials & Assignments**: Access and download resources uploaded by professors (PDFs, reference links, etc.).
- **Exam Results**: View performance and grades for past examinations (e.g., Mid-Semester results).

### Teacher Dashboard
The teacher dashboard empowers educators with tools to manage their classes effectively:
- **Overview Dashboard**: View upcoming classes with live status indicators, access quick actions, and read the latest notices.
- **Class & Student Management**: View lists of enrolled students for specific classes and manage enrollments (add/remove students).
- **Attendance Management**: Mark and update daily attendance for students (Present, Absent, Late).
- **Material Distribution**: Upload study materials and resources for students to access.
- **Marks Publishing**: Post examination marks by uploading CSV files.
- **Progress Reports**: Generate comprehensive progress reports (Attendance, Exam Marks, or both) in PDF or Excel formats.

## Technologies Used
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Icons & Fonts: FontAwesome, Google Fonts (Inter)
- Backend: Node.js, Express (API structure setup)
